# Atlas Core Ubuntu Icon Assets

Generated from the approved ATLAS CORE green/black app icon mockup.

## Recommended project path

Put the 512px icon here:

```text
ui/assets/atlas-core.png
```

Or for a Linux desktop app package:

```text
/usr/share/icons/hicolor/512x512/apps/atlas-core.png
```

## Desktop entry example

```ini
[Desktop Entry]
Name=Atlas Core
Comment=Local AI command center
Exec=/path/to/atlas-core
Icon=atlas-core
Terminal=false
Type=Application
Categories=Development;Utility;
```

## Codex instruction

Use `atlas-core.png` as the application icon and header logo source.
For the UI header, display it beside the `ATLAS CORE` wordmark or centered above it.
Do not use the full mockup image as the app icon; use the cropped square icon asset.
